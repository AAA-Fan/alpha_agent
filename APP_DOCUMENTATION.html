Financial Agent App Documentation

Comprehensive guide for setup, running, API usage, dashboard monitoring, and troubleshooting.

Project path: /home/uniconsoft/Downloads/financailagent (1)/financailagent

CLI FastAPI Streamlit Multi-Agent Forecast + Risk + Backtest

1. What This App Does

This application analyzes a stock ticker through a multi-agent pipeline and produces a final recommendation with both qualitative and quantitative evidence.

  • Historical analysis and indicator analysis (RSI logic)
  • News sentiment analysis
  • Pair momentum divergence monitoring
  • Feature engineering and regime classification
  • Probabilistic forecast, risk plan, and backtest snapshot
  • Supervisor synthesis and final report

2. Main Components

  • main.py: CLI runner
  • orchestrator.py: shared 10-step pipeline
  • api.py: FastAPI backend
  • dashboard.py: Streamlit frontend
  • agents/: all analysis and decision agents
  • utils/storage.py: SQLite/Postgres persistence

3. Output Artifacts

  • Text report files: report_<SYMBOL>_<TIMESTAMP>.txt
  • SQLite tables (default): recommendations, predictions, pair_signals, realized_outcomes
  • RSI and history csv snapshots in data/
  • Optional trained forecast model: data/forecast_model.json

4. Prerequisites

  • Python 3.11 recommended
  • Conda (recommended for isolated environment)
  • OPENAI_API_KEY required
  • ALPHAVANTAGE_API_KEY required
  • TAVILY_API_KEY optional

5. Environment Setup (Recommended)

5.1 Create and activate Conda env

conda create -y -n finagent python=3.11
conda activate finagent

5.2 Install dependencies

cd "/home/uniconsoft/Downloads/financailagent (1)/financailagent"
pip install -r requirements.txt

5.3 Configure environment variables

cp .env.example .env

Update at least these keys in .env:

OPENAI_API_KEY=...
ALPHAVANTAGE_API_KEY=...
TAVILY_API_KEY=...   # optional

6. How to Run the App

6.1 Start backend API (terminal 1)

conda activate finagent
cd "/home/uniconsoft/Downloads/financailagent (1)/financailagent"
uvicorn api:app --host 0.0.0.0 --port 8000

6.2 Start frontend dashboard (terminal 2)

conda activate finagent
cd "/home/uniconsoft/Downloads/financailagent (1)/financailagent"
streamlit run dashboard.py

6.3 Optional CLI run (without frontend)

conda activate finagent
python main.py

7. Frontend Validation Checklist (How to Know It Works)

  1. Open dashboard URL shown by Streamlit (usually http://localhost:8501).
  2. Sidebar: confirm Use API backend is ON by default.
  3. Configuration tab: verify API health shows ok.
  4. Live Analysis tab: run ticker AAPL.
  5. Confirm progress completion and populated metric cards (Action, Prob. Up, Predicted Return, Position Size).
  6. Check Final Recommendation section has detailed output text.
  7. Monitoring tab: confirm row counts and charts update (recommendations, predictions, pair signals).
  8. Run second ticker (e.g., MSFT) and verify counts increase.
If counts remain zero, verify STORAGE_ENABLED=true and STORAGE_URL points to writable storage.

The system runs without a trained forecast model by using a fallback heuristic. To generate a local model config: 

conda activate finagent
python pipelines/train_forecast_model.py

Generated file: data/forecast_model.json

11. Troubleshooting

Problem: API does not start

  • Check dependencies: pip install -r requirements.txt
  • Try another port: uvicorn api:app --port 8010
  • Verify env keys in .env

Problem: Dashboard can open but no data

  • Ensure backend is running and URL in sidebar is correct
  • Verify Use API backend is ON
  • Run one analysis first to populate tables

Problem: Import errors around LangChain APIs

  • Use pinned versions in requirements.txt
  • Reinstall inside clean env: pip install -r requirements.txt

12. Operational Notes

  • API backend is default-enabled in dashboard UI.
  • Storage reads in API currently target SQLite endpoints for monitor routes.
  • External API quotas (OpenAI, Alpha Vantage, Tavily) can affect speed and completeness.
